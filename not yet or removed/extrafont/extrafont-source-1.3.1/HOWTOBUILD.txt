HOW TO BUILD:
-------------

You should repeat the build procedure on different platforms:
  Windows
  Linux
  MacOSX
At the end, you will find all the results in a directory named 'XBUILD/extrafont'
ready for the distribution  

* NOTE* starting with version 1.3.1 , support for 32-bit is no more avalible.

Windows:
--------
Using MS Visual Studio

  ms-build tcl-8 x32
  ms-build tcl-8 x64
  ms-build tcl-9 x64
  
Linux
-----  
Using the standard GNU/C tool-chain   

  build.sh tcl-8 x32
  build.sh tcl-8 x64
  build.sh tcl-9 x64
  
MacOSX
------  
Using the standard GNU/C tool-chain   

  build.sh tcl-8 x64
  build.sh tcl-9 x64
